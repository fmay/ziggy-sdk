# Ziggy Plugin SDK

Official SDK for developing custom block plugins for Ziggy Services platform.

## Overview

The Ziggy Plugin SDK enables external developers to create custom workflow blocks with:
- ✅ Full TypeScript support
- ✅ React components for UI
- ✅ Server-side execution logic
- ✅ Browser-compatible distribution
- ✅ Hot-reload during development

## Installation

```bash
npm install -g github:fmay/ziggy-sdk
```

## Documentation
This SDK is for use with Ziggy Plugins. See [Plugin documentation](https://docs.ziggyservices.com/user-guide/plugin/overview)
